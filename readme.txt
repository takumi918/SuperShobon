このたびは、スーパーしょぼんをダウンロードしていただき、ありがとうございます


[このゲームについて]

このゲームはJAVAアプレットです。アプレットの環境があれば 実行することができると思います


[ゲームをプレイする方法]

Main.htmlを実行し、アプレット起動部分を選択すれば、動かせます


[内容の編集]

主に名前さえ変更しなければ、問題は起きません。
一部のファイルではconfigにて名前の参照がありますので  そちらを変更すれば、好きな画像、音楽等に変更できます


また、ステージ作成にてエディタを使用すると簡単に作成できます

http://www.geocities.jp/z_gundam_tanosii/applet/super-syobon/main.html



[編集したゲームの公開]

最低でも、このreadme.txtにあったフォルダ (中身はあるものとする)

stage
res
config

及び、

Main.class 
Main.html

を 同じフォルダにアップロードし、Main.htmlを公開することで、編集したゲームをプレイすることができます


[配布に関して]

転載可能、許可不要です。二次配布等もいっこうに構いません。
有償は不可です (アフリエイト等の広告は除く)
